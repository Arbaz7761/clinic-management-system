## 📌 Project Overview

This is a data-driven clinic management system designed to manage patient records, appointments, and healthcare operations efficiently.

---

## 🚀 Features

* Patient registration and management
* Appointment scheduling
* Medical records tracking
* Secure login system
* Real-time search and dynamic updates
* Data visualization using charts

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, Bootstrap, JavaScript
* Backend: Python, Flask
* Database: SQLite, SQL
* Data Tools: Pandas, Chart.js
* Deployment: PythonAnywhere
